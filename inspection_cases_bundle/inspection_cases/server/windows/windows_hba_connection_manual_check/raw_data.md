# 영역
HBA

# 세부 점검항목
HBA 연결상태 점검

# 점검 내용
HBA 연결 상태 수동 확인 필요 여부 기록

# 구분
필수

# 명령어
수동 확인

# 출력 결과
```text
수동 확인 대상: HBA 연결 상태는 벤더 HBA 도구 또는 SAN 관리도구와 교차 확인해야 합니다.
```

# 설명
- Windows OS 기본 명령만으로 신뢰성 있게 자동 판정하기 어려운 항목이다.
- 벤더 관리도구, OOB 관리 콘솔, SAN 관리도구 또는 현장 확인 결과와 교차 확인한다.

없음
