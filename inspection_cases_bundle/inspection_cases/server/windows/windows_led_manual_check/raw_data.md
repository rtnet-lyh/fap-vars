# 점검 내용
장비 LED 상태 수동 확인 필요 여부 기록

# 구분
필수

# 명령어
수동 확인

# 출력 결과
```text
수동 확인 대상: LED 상태는 원격 Windows OS 명령으로 신뢰성 있게 판정할 수 없어 현장 또는 OOB 관리도구 확인이 필요합니다.
```

- 벤더 관리도구, OOB 관리 콘솔, SAN 관리도구 또는 현장 확인 결과와 교차 확인한다.
