# 영역
DISK

# 세부 점검항목
파일시스템 사용량

# 점검 내용
파일시스템 사용량 점검

# 구분
필수

# 명령어
```bash
bdf
```

# 출력 결과
```text
Filesystem          kbytes    used   avail %used Mounted on
/dev/vg00/lvol3   20971520  8300000 12671520  40% /
/dev/vg00/lvol4   52428800 42000000 10428800  80% /opt
/dev/vg00/lvol5  104857600 70000000 34857600  67% /var
tmpfs              8192000        0  8192000   0% /dev/shm
```

# 설명
- `bdf` 명령으로 HP-UX 파일시스템별 사용량과 가용 용량을 확인한다.
- 사용률이 높거나 가용 공간이 부족한 파일시스템은 로그 정리, 임시 파일 삭제, 용량 증설을 검토한다.
- `/proc`, `/dev` 등 운영 용량 판단 대상이 아닌 특수 파일시스템은 제외할 수 있다.
- 명령 실행 불가, 권한 문제, 출력 파싱 실패 시에는 확인 필요로 분류한다.

# 임계치
USED_MAX_PCT
AVAIL_MIN_PCT
IGNORE_MOUNTS

# 판단기준
- **양호**: 운영 대상 파일시스템 사용률이 `USED_MAX_PCT` 미만이고 가용률이 `AVAIL_MIN_PCT` 이상인 경우
- **경고**: 사용률이 `USED_MAX_PCT` 이상이거나 가용률이 `AVAIL_MIN_PCT` 미만인 파일시스템이 있는 경우
- **확인 필요**: `bdf` 결과를 확인하거나 해석할 수 없는 경우
